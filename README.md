Учебный сайт магазина одежды (fastAPI)

Запуск:
1) pip install -r requirements.txt
2) uvicorn app.main:app --reload

Приложение включает:

✅ Полнофункциональный фронтенд

✅ Адаптивный дизайн

✅ Корзину покупок

✅ Систему логирования

✅ Базу данных SQLite

✅ API документацию (/docs)

✅ Страницы: главная, товары, корзина, вход, регистрация
